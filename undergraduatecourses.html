Undergraduate Courses

I took undergraduate courses in computer science on my path to being accepted into the master's program in computer science. I feel that the undergraduate courses that I took as prerequisites for graduate school gave me a sufficiently broad understanding of computer science to succeed and I enjoyed learning the concepts.

Introduction to Computer Science and Object-Oriented Programming

We practiced test-driven development in Java using JUnit. This was a good introduction to object-oriented programming and Java.

Object-Oriented Data Structures and Algorithms

This course emphasized data structures, algorithms, and design patterns using Java. We continued to use JUnit for testing.

Principles of Computer Organization

This course emphasized computer hardware and assembly language. I enjoyed using the little man computer was used to learn assembly language. The trick was that we had a limited number of lines to accomplish specific tasks using simple commands.

Database Management

This course was a broad survey of database management and there was a lot to remember. We programmed using SQL on an Oracle databse.

Operating Systems

This was a broad survey of the theory behind operating systems. We used the C programming language for labs along with Pthreads. This course was also used to improve our understanding of Linux. I enjoyed the labs and found myself thinking more about algorithms than I thought I would in this course.

Discrete Mathematics

The math covered this course was helpful for analyzing algorithms and for understanding relational databases.
